# CAN-DBCs
Repository for CAN DBCs (CAN Databases)

These DBCs were obtained form the roof filesystem of a shipped tractor, and are being presented as an effort to promote right-to-repair efforts of these abandoned tractors.  We believe that this qualifies under 37 CFR § 201.40 as an exemption to DMCA copyright enforcement provisions.  (But we are not lawyers, so don't sue us please!)

# More Information: 
https://www.ecfr.gov/current/title-37/chapter-II/subchapter-A/part-201/section-201.40

Obviously we can't confirm the accuracy of these, use at your own risk!
